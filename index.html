Game Grid is a video-game discovery website for creating lists and learning about various video-games. Users can create an account and begin adding games to their wishlists or custom lists.


This is a full-stack website that I developed as a personal project to practice and gain additional knowledge and experience. This website offers a wide variety of video-games across almost all gaming platforms to be browsed. The user can create their account through the sign up page as their information will be stored in a MongoDB database and the passwords protected using hashing and salting.

The most challenging part for me during the development of this website was the task of listing the video-games the use has added to their wishlist onto the lists webpage. Fortunately, I overcame this obstacle by creating additonal variables in the MongoDB database such as the video-game name and the image url.

Let Me Cook is a cooking recipe website for finding your next meal to prep and calculating your calorie intake of the week.


This website is a university project that tasked us to develop and desing a website using basic HTML, CSS, and JavaScript concepts and create a front-end website with a great visual. The concept of Let Me Cook is to allow its users to browse a variety of recipes to then add them to their weekly schedule to calculate the calorie intake of the current week.

The most time-consuming feature to develop in this website was creating the schedule to add add the appropriate amount of calories to the calories total and remove the recipes by simply double clicking the cells of the table. However, I was able to overcome this challenge by using switch case in the JavaScript code.
